# 🚀 Claude Code Skills Arsenal (claudinho-skills)

**Eleve o Claude Code ao nível de Staff Engineer, Arquiteto e C-Level.** *Um instalador automatizado com 60 personas especialistas e diretivas arquiteturais de alto nível para revolucionar sua produtividade no terminal.*

---

## 📖 Visão Geral

O **Claude Code Skills Arsenal** é um instalador automatizado (`install-claude-skills.sh`) projetado para injetar 60 *skills* (agentes de contexto) customizadas diretamente na configuração local do seu Claude Code (`~/.claude/skills`).

Cada skill atua como uma injeção profunda de contexto, transformando o LLM padrão em um especialista ultra-focado com metodologias rigorosas, padrões arquiteturais modernos e restrições de qualidade (guardrails) pré-definidos. Em vez de escrever prompts longos a cada interação, você simplesmente invoca o especialista correto para o desafio atual.

## ✨ Características Principais

* **Instalação Zero-Friction:** Um único script Bash idempotente que cria a estrutura completa de pastas e arquivos `.md` instantaneamente.
* **Engenharia de Prompt Avançada:** Cada skill foi desenhada utilizando frameworks reais de mercado (SOLID, Clean Architecture, DDD, Core Web Vitals, RICE, Kano Model, etc.).
* **Padronização Estrita:** Todas as saídas são forçadas a seguir estruturas analíticas de alta senioridade, eliminando respostas superficiais ou generalistas.
* **Ampla Cobertura:** 12 categorias distintas cobrindo todo o ciclo de vida do desenvolvimento de software, desde a ideação do produto até o escalonamento FinOps na nuvem.

## 🛠️ Instalação

A instalação leva menos de 5 segundos. Nenhuma dependência externa é necessária, apenas um terminal Bash.

1. **Faça o download do script:**
Copie o arquivo `install-claude-skills.sh` para a sua máquina local.
2. **Dê permissão de execução:**
```bash
chmod +x install-claude-skills.sh

```


3. **Execute o instalador:**
```bash
./install-claude-skills.sh

```



Você verá a confirmação de que todas as 60 diretivas foram instaladas em `~/.claude/skills`.

## 🚀 Como Utilizar

Inicie o Claude Code normalmente no diretório do seu projeto. Para invocar qualquer um dos especialistas, use o comando `/` seguido do nome da skill e, em seguida, descreva seu problema ou cole seu código.

**Exemplo de Refatoração:**

> `/code-refactorer analise o arquivo src/services/PaymentGateway.ts. Ele está acoplado e violando o Princípio de Responsabilidade Única.`

**Exemplo de Debugging Profundo:**

> `/debugger o endpoint /api/webhooks/stripe está retornando 500 intermitentemente. Aqui estão os logs das últimas 3 falhas: [COLE OS LOGS]`

**Exemplo Arquitetural:**

> `/system-architect proponha uma arquitetura de microsserviços orientada a eventos para um e-commerce que precisa processar 10.000 pedidos por minuto na Black Friday.`
